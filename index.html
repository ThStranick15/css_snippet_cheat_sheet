Flex Row

Use these properties to create a flex row layout.

 .row {
            display: flex;
            flex-direction: row;
            flex-wrap: wrap;
        }

Flex Column

Use these properties to create a flex column layout.

 .column {
            display: flex;
            flex-direction: column;
        }

Grid

Use these properties to create a 12 column grid layout.

 .grid {
            display: grid;
            width: 100%;
            grid-template-columns: 
            repeat(12, 1fr);
        }
